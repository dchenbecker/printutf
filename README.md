README
======

This is a script I threw together in disgust at the state of printing from
Mutt. It's highly specialized for my system, with the following dependencies:

* xelatex, with the following packages:
  * ucharclass
  * fontspec
  * geometry
  * fancyhdr
  * lastpage
* formail
* zsh
* pdfnup
* The following fonts:
  * DejaVu Sans Mono
  * AR PL UKai TW
